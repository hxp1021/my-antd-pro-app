# Ant Design Pro

使用 [Ant Design Pro](https://pro.ant.design) 脚手架工具初始化的项目模板，具体开发指南可以参考 [Antd 官方文档](https://pro.ant.design/zh-CN/docs/overview)。

### 注意

此模板中使用了 Antd 官方的后端示例服务 https://proapi.azurewebsites.net ，开发者可以根据需要改为自己的后端服务。

### 安装依赖

```bash
npm install
```

### 启动项目

```bash
npm start
```

### 构建项目

```bash
npm run build
```
